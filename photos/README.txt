PHOTOS FOLDER
=============

Drop your images in here:  .jpg  .jpeg  .png  .webp  .gif  .svg  .bmp  .avif

They appear automatically in the ambient slideshow — the folder is re-read every
time the slideshow starts, so no restart is needed for new photos.

This folder ships with three sample gradient images so the slideshow works on the
very first run. Delete them (sample-*.svg) once you've added your own.

Prefer to keep photos somewhere else (e.g. ~/Pictures)? Change this line in
config/config.js:

    ambient: { photoDirectory: "/home/athervr/Pictures", ... }
